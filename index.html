🔥 إعداد Firebase
ربط البرنامج بالسحابة للتزامن بين أجهزتك
📋 الصق هنا بيانات مشروعك من Firebase Console.
لا تعرف كيف؟ اضغط تبويب "دليل الإعداد" أعلاه.
🔐 سجّل دخولك بالبريد الذي أنشأته في Firebase Authentication.
إذا لم تُنشئ حساباً بعد، اضغط "إنشاء حساب جديد".
📖 خطوات إعداد Firebase من الصفر
① افتح Firebase Console
اذهب إلى: console.firebase.google.com
سجّل دخولك بحساب Google (أي حساب Gmail لديك)
② أنشئ مشروعاً جديداً
• اضغط "Add project" أو "إضافة مشروع"
• الاسم: bassam-teacher
• أوقف Google Analytics (اختياري)
• اضغط "Create project"
③ أضف تطبيق Web
• من الصفحة الرئيسية للمشروع اضغط أيقونة </>
• الاسم: bassam-app
• اضغط "Register app"
• انسخ كامل الـ firebaseConfig وألصقه في تبويب "إعداد المشروع"
④ فعّل Firestore Database
• من القائمة الجانبية: Build → Firestore Database
• اضغط "Create database"
• اختر "Start in test mode" (للبدء السريع)
• اختر أي منطقة (مثلاً: europe-west1)
• اضغط "Enable"
⑤ فعّل Authentication
• من القائمة: Build → Authentication
• اضغط "Get started"
• اختر Email/Password
• فعّله ✅ واضغط Save
• اذهب لـ Users → Add user
• أدخل بريدك وكلمة مرور واحفظها
⑥ ارجع هنا وأدخل بياناتك
• اضغط تبويب "إعداد المشروع" والصق الـ config
• اضغط تبويب "تسجيل الدخول" وادخل بريدك وكلمتك
• ✅ انتهى! بياناتك ستُزامَن على كل أجهزتك
